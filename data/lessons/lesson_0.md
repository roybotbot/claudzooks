# Lesson 0: Prerequisites

Before we start, let's make sure your computer is set up.

## Python
You need Python 3.11 or newer. If you're seeing this message,
Python is already installed — the game wouldn't run otherwise. ✓

## Claude Code (needed for Lessons 6-7)
Claude Code is an AI coding tool that runs in your terminal.
You don't need it yet, but you will for the final two lessons.

To install it later:
1. You need a Claude Pro, Max, or Team subscription at claude.ai
2. Open your terminal and run:

       curl -fsSL https://claude.ai/install.sh | bash

3. Restart your terminal (close and reopen it)
4. Run: claude
5. Follow the prompts to log in

Don't worry about this now. We'll check again before Lesson 6.
